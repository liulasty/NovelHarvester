# m.shuwen6.cc 书文小说网（`scrape-shuwen6.js`）

与 book18 流程一致：分章进 `chapters/`，合并进 `merged/`（有 `--merge-title` 时以书名为文件名，否则为 `全文合并.txt`）。

## 目录分页（多页目录）

- 同一本书的目录常按 **约 90 章/页** 拆成多页（例如 `ml.html` → `ml_2.html` → `ml_3.html`，共 3 页、合计 205 章）。**仅第一页就有 90 个章节标题属正常现象**；必须把 **所有目录分页** 都抓完，不能只认第一页。
- 脚本会识别 **指向 `…/ml.html` / `…/ml_N.html` 的「下页」**（与章节阅读页的「下页」区分），依次打开每一页目录；解析时会在控制台打印每页链接条数（如 90、90、25）。

## 目录列表

- Selector：**`#list ul.chapter-list`**。
- 每页加载完后再用 `waitForFunction` 确保列表内至少有一条章节链接，避免末页加载慢导致漏章。

## 正文页

- 正文在 **`#chapter-content`**。页面先出现「章节内容加载中」，再由前端注入真实内容。
- 为避免广告/长连接导致 **`networkidle` 永不达成**（进而 `page.goto` 超时），脚本使用 **`domcontentloaded` / `load` 分层导航**（`gotoChapterPage`），再 **`waitForSelector('#chapter-content')`**，并等待正文中不再出现「章节内容加载中」。

## 正文解码

- 优先从页面解析 **`initTxt("…/xx.wen")`**，请求 `.wen` 并在 VM 中执行 **`_txt_call`**（与站点一致）；失败或过短则回退 DOM。
- DOM 路径下敏感字常以 **`<img class="TImg">`** 插入，需在 **`timg-map.json`**（与本文件同目录）用 **完整 `src` 字符串** 映射到单字；未映射处正文用 **`□`** 占位，控制台会提示未映射数量。

## 章内分页

- 一章可能多屏，阅读页也有 **「下页」**（同章路径 + `?page=2`）。
- 脚本只取 **与当前章节 path 相同** 的「下页」链接，避免与目录分页或其它「下页」混淆；拼满同章后再进下一章。

## 失败记录 `failed_chapters.json`

输出目录下可生成 **`failed_chapters.json`**（可用 **`--failed-log=`** 改文件名或绝对路径），当前格式为 **`version: 2`**：

| `failures[].kind` | 含义 |
|-------------------|------|
| `extraction_error` | 整章抓取抛错（如超时、`page.goto` 失败等） |
| `timg_unmapped` | 分章已写入，但仍有未在 `timg-map.json` 中映射的 TImg；含 **`unknownTimg`** |

**仅重跑记录中的章节**（不重新扫目录），并 **强制覆盖已有分章文件**（便于补映射后重拉）：

```bash
node gaode/shuwen6/scrape-shuwen6.js --out-dir=novel-output/<书籍目录> --retry-failed
node gaode/shuwen6/scrape-shuwen6.js --out-dir=novel-output/<书籍目录> --retry-failed --merge
```

全量再跑时，若某章因「本地已存在」被跳过，**不会**抹掉该章在 `failed_chapters.json` 里已有的 **`timg_unmapped`** 记录（除非该章被重新抓取且已无未映射 TImg）。旧版记录无 `kind` 字段时，视为 **`extraction_error`**。

## 使用 @playwright/cli 做页面研究

依赖中的 `npx playwright-cli` 可在终端驱动浏览器，便于确认选择器。典型用法（书文 m 站）：

```bash
npx playwright-cli open
npx playwright-cli goto https://m.shuwen6.cc/xs/AjMk/ml.html
npx playwright-cli snapshot
npx playwright-cli eval "() => document.querySelectorAll('#list ul.chapter-list li a').length"
npx playwright-cli goto https://m.shuwen6.cc/xs/AjMk/1.html
npx playwright-cli eval "() => document.querySelector('#chapter-content')?.innerText?.slice(0, 500)"
```

目录或正文请以 **`snapshot` / `eval`** 核对当前 DOM，再写进 `scrape-shuwen6.js`。

## 环境变量（可选）

- **`SHUWEN6_ML_URL`**：`scrape-shuwen6.js` 未传参时的目录页 URL（一般直接用命令行传入即可）。

## 参考文件

- **`reference/`**（可选）：从书文站保存的页面脚本片段（如 `_chapter.js`），仅供对照 `.wen` / 前端逻辑，**不参与**运行时 `require`。

